INSTALLATION VIEW

Suck #[1-5], 2021
dimensions variable
five unique vacuumed piles of dirt, each pile separated in one of the five same handheld vacuum cleaner

The installation is five of the same handheld vacuum cleaner, all sitting on the floor in a box room.
Each separate handheld vacuum cleaner contains the dirt of a person that used the service, Dirty Work (2021).
